# dotfiles

It was my configuration for Arch Linux, managed by
[chezmoi](https://www.chezmoi.io/).

## Notice

chezmoi is great, but Nix was more convenient for me in the end. My new
configuration is [system (on codeberg)](https://codeberg.org/shimeoki/system),
and I still use it on Arch Linux (at least at the moment of archiving).

You are, of course, still free to still use the code from here, but don't try to
install the configuration with `chezmoi apply`. Also be aware that most settings
here are probably outdated.
